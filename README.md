# 🍕 Pizza Time

Eine responsive Restaurant-Bestellanwendung, entwickelt mit HTML, CSS und JavaScript.

## 🌐 Live Demo

👉 https://janetbughardt.github.io/pizza-time/

## 📖 Projektbeschreibung

Pizza Time ist eine interaktive Restaurant-Webanwendung. Nutzer können Gerichte auswählen, in den Warenkorb legen und den Gesamtpreis inklusive Lieferkosten automatisch berechnen lassen.

## ✨ Funktionen

- Dynamisches Menü
- Warenkorb
- Produkte hinzufügen und entfernen
- Automatische Preisberechnung
- Zwischensumme
- Lieferkosten
- Gesamtsumme
- Responsive Design für Desktop und Mobilgeräte

## 🛠️ Verwendete Technologien

- HTML5
- CSS3
- JavaScript (ES6)
